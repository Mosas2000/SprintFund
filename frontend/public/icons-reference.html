SprintFund Icons Reference

Available Icon Files

Favicon

  • favicon.ico - Multi-resolution ICO (16x16, 32x32, 48x48)
  • favicon.svg - SVG favicon for modern browsers
  • favicon-16x16.png - 16x16 PNG
  • favicon-32x32.png - 32x32 PNG

PNG Icons (Various Sizes)

  • icon-16.png - 16x16
  • icon-32.png - 32x32
  • icon-48.png - 48x48
  • icon-72.png - 72x72
  • icon-96.png - 96x96
  • icon-128.png - 128x128
  • icon-144.png - 144x144
  • icon-152.png - 152x152
  • icon-180.png - 180x180
  • icon-192.png - 192x192 (PWA standard)
  • icon-196.png - 196x196
  • icon-256.png - 256x256
  • icon-384.png - 384x384
  • icon-512.png - 512x512 (PWA standard)

Apple Touch Icon

  • apple-touch-icon.png - 180x180 for iOS devices

Open Graph

  • og-image.png - 1200x630 for social media sharing
  • og-image.svg - SVG version

SVG Icons

  • icon.svg - Base SVG icon (512x512)

Configuration Files

  • manifest.json - PWA manifest
  • site.webmanifest - Web app manifest with shortcuts
  • browserconfig.xml - Windows tile configuration

Usage in HTML

<!-- Favicon -->
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/icon.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">

<!-- Manifest -->
<link rel="manifest" href="/manifest.json">

<!-- Open Graph -->
<meta property="og:image" content="/og-image.png">
