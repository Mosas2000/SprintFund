# Stake Components

This directory contains components for displaying stake lock status and related information.

## Components

### StakeDashboard
Main dashboard showing comprehensive stake information.

### StakeLockStatus
Displays breakdown of total, locked, and available stake.

### LockedStakeBreakdown
Shows detailed information about locked funds per proposal.

### StakeLockWarning
Inline warning component for locked funds.

### CompactStakeStatus
Compact display for use in headers or sidebars.

## Usage

See `frontend/docs/STAKE_LOCK_SYSTEM.md` for detailed documentation.
